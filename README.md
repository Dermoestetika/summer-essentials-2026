# D&F Summer Essentials 2026

Event management aplikacija za D&F Summer Essentials seriju događaja.

## Komponente

- **Admin Panel** (`admin.html`) - Upravljanje učesnicima, eventima, sponzorima
- **Mobile App** (`app.html`) - PWA aplikacija za učesnike (kvizovi, rang lista)
- **Check-in Tablet** (`checkin.html`) - QR skeniranje na ulazu
- **TV Leaderboard** (`leaderboard.html`) - Projektor/TV prikaz rang liste

## Deploy

### Railway (preporučeno)

1. Forkaj ovaj repo na GitHub
2. Idi na [Railway](https://railway.app)
3. New Project → Deploy from GitHub
4. Odaberi repo

### Lokalno

```bash
npm install
npm start
```
